# 🏗️ Buildings Models Directory

Place your GLB model files here for parcel structures.

## Files to Add

- `tower.glb` - Tower/Office building structure
- `gallery.glb` - Gallery/Museum piece
- `shop.glb` - Retail shop facade
- `mansion.glb` - Large mansion/house

## How to Get Models

See: `GLB_MODELS_QUICK_START.md` in project root

**Quick Option:**
1. Go to: https://sketchfab.com
2. Search: "free building glb"
3. Download GLB format
4. Rename to match files above
5. Drop in this folder

**Models are loaded automatically** when placed here!
